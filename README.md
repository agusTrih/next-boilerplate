## Quick Start

Use Node 14 atau lebih
Use Yarn 1.22 atau lebih

```javascript 
"engines": {
    "node": ">=14.0.0",
    "yarn": ">=1.22.0",
    "npm": "please-use-yarn"
  },
```

Clone the project

```bash
  git clone https://gitlab.com/Agustrihanton/demfarm
```

Go to the project directory

```bash
  cd personal-website
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```

for commit please see file 
``commitlint.config.js``
